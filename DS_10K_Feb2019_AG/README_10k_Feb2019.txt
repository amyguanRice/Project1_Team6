This is the README file for the dataset: 10K Data Science Jobs in Feb 2019.

The 'data_processing_10k_Feb2019.ipynb' file includes processes to conduct the following tasks:

	1) data cleaning
	2) industry categorization
	3) grouped job title analysis (by junior/senior levels)
	4) grouped job type analysis (by full-time/part-time/contractor)
	5) desired technical skills/technical tools/soft skills for all industries
	6) break-down of desired technical tools by industry

The 'Geographic_Dist_10k_Feb2019.ipynb' file includes the code to calculate and visualize the geographical distribution
